# Jig Label UI

## How to install

```sh
yarn install
```

## How to build

```sh
# Builds the project and opens it in your browser.
# It will auto-reload when you make any changes.
yarn start
```

## How to build for production

```sh
# Builds the project and places it into the `dist` folder.
yarn run build
```
